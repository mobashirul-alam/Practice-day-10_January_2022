Semantic Tags

A semantic element clearly describes its meaning to both the browser and the developer. These are part of HTML5. In semantic tag, it is easy to identify what kinds of information the tag contains. Semantic tags are easier to read and understand; both for the coder and search engine. Here's the list of semantic tags:

  • article
  • aside
  • details
  • figcaption
  • figure
  • footer
  • header
  • main
  • mark
  • nav
  • section
  • summary
  • time
